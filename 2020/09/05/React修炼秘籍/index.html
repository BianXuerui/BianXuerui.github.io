7.React生命周期

在某一时刻，可以自动执行的函数

1.Initialization(初始化阶段)

初始化props和state阶段

2.Mounting(虚拟DOM挂载阶段)
  1. componentWillMount （DOM挂载之前执行的）
  2. render （state或者props改变时执行）
  3. componentDidMount （DOM挂载完了执行）
3.Updation(更新阶段)
  1. props
    1. componentWillReceiveProps
      1. 组件第一次存在于DOM中，函数是不会被执行的
      2. 如果已经存在于DOM中，函数才会被执行
    2. shouldComponentUpdate
      1. 要求必须返回一个布尔值
      2. 如果返回true生命周期会继续执行，返回false生命周期将不会再继续执行
      3. 在优化组件性能有大用处
    3. componentWillUpdate
    4. render
      1. 必须写
    5. componentDidUpdate
      1. DOM更新渲染完再执行
  2. state
    1. shouldComponentUpdate
    2. componentWillUpdate
    3. render
    4. componentDidUpdate
4.Unmounting(卸载DOM阶段)

当我们的组件将要被删除，还没有被删除的时候执行
