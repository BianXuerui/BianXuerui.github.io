基于Hexo搭建博客

2020-04-01 648

环境：Node.js、npm

全局安装cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo

cnpm install -g hexo-cli

初始化hexo项目

hexo init

hexo基本命令

hexo n(new) 生成一篇新的文章
hexo c(clean)
hexo g
hexo s(serve)

hexo部署github

新建一个仓库名称:必须符合特定要求才行  username.github.io

git部署插件安装

cnpm install --save hexo-deployer-git

修改文件_config.yml

deploy
  type: git
  repo: git仓库地址
  Branch：master（不写默认master）

输入hexo d 部署项目

git输入账号密码
账号是用户名不是邮箱！！！

更换主题（推荐yilia主题）

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
进入根目录下_config.yml文件修改theme主题 注意theme：后面有一个空格！
hexo c (clean) && hexo g (generate) && hexo s (server) 
hexo d(deployer) 推送到github上
