基于Hexo搭建博客

2020-04-01 687
环境：Node.js、npm
全局安装cnpm
安装hexo
初始化hexo项目
hexo基本命令
  1. hexo n(new) 生成一篇新的文章
  2. hexo c(clean)
  3. hexo g
  4. hexo s(serve)
hexo部署github
新建一个仓库名称:必须符合特定要求才行 username.github.io
git部署插件安装
修改文件_config.yml
输入hexo d 部署项目
  1. git输入账号密码
  2. 账号是用户名不是邮箱！！！
更换主题（推荐yilia主题）
  1. git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  2. 进入根目录下_config.yml文件修改theme主题 注意theme：后面有一个空格！
  3. hexo c (clean) && hexo g (generate) && hexo s (server)
  4. hexo d(deployer) 推送到github上
